# Vault Finanzas — Hi-fi design

A high-fidelity visual design for a household finance control system. Spanish copy, dark theme, side-by-side screens on a design canvas.

## Screens
1. **Dashboard** — true available balance, cash flow forecast, KPIs
2. **Transactions Ledger** — searchable, filterable, with ghost charges flagged
3. **Reconciliation Workspace** — import review, duplicate detection, coverage by source
4. **Budgets** — category envelopes with smart pacing
5. **Review Queue + Shared (couples)** — items to review and shared finances
6. **Smart Calendar** — payment scheduling + future bills

Tweaks: theme, accent, density.
